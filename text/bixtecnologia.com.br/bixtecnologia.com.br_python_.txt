 Uma linguagem de programação completa para transformação digital e projetos de ciência de dados. 














  


















O QUE É O PYTHON? 










Lançado em 1991, o Python é uma linguagem de programação de alto nível. É muito popular entre os profissionais de tecnologia da informação por conta do seu grande número de aplicações e por ser open source e desenvolvido de forma comunitária. É gerenciado pela Python Software Foundation, uma organização sem fins lucrativos.Com linguagem rápida, grande potencial de escalabilidade, estrutura e sintaxe eficientes e intuitivas, a participação do Python em projetos que envolvam Análise e Ciência de Dados vem crescendo muito nos últimos anos. Dentre as próprias várias aplicações possíveis, as bibliotecas relacionadas à Data Analysis têm atingido importante grau de maturidade.Recursos relacionados a visualização de dados estão presentes no Python, permitindo aos usuários criar dashboards e gráficos interativos. Por conta disso, as informações que uma empresa possui se tornam muito mais acessíveis e democráticas, fazendo com que profissionais de diversos setores consigam começar a tomar decisões e traçar estratégias com base em dados. Além disso, o uso do Python possibilita ao profissional, por exemplo,  minerar e modelar dados, usar Machine Learning e desenvolver sistemas web. Essas são ações necessárias e fundamentais em projetos de ciência de dados.
