SOLUÇÕES DE SISTEMAS EM PROJETOS 


















 



Application Programming Interface (APIs) 



APIs são um conjunto de padrões que compõem uma interface. São elas que permitem o transporte de dados entre um cliente e um servidor. As APIs possibilitam a criação de plataformas de modo ágil e simplificado para os desenvolvedores. É possível criar softwares, aplicativos, programas e plataformas diversas. 







 



Web Scraping 



Uma técnica que permite extrair dados para análise de modo automatizado, utilizando bots e possibilitando a coleta de uma quantidade expressiva de dados em pouco tempo. As informações geradas a partir do web scraping podem ser utilizadas para embasar tomadas de decisão de forma assertiva e inteligente. 











 



Desenvolvimento Web 



Criação de um software, seja sistema, aplicativo ou site, que está conectado a uma rede. Hoje, a maior parcela do que é desenvolvido é feito para web, permitindo conectividade, integração entre diferentes softwares e ações com resultados instantâneos. 














 



Desenvolvimento Mobile 



O desenvolvimento mobile é destaque no mercado por conta da acessibilidade aos dispositivos móveis, como os smartphones. Essa é uma ótima opção para o uso cotidiano devido à portabilidade e a praticidade.

FERRAMENTAS UTILIZADAS 


















 







FastAPI 



Uma ferramenta moderna e de alta performance para a construir APIs com Python 3.6+.É um dos mais velozes frameworks de Python, tornando o processo de codificar mais rápido e reduzindo as possibilidades de erros. 


















 







Django 



Um framework web full stack open source (código aberto) de alto nível e baseado em Python. Foi desenvolvido para solucionar problemas em aplicações web, suporta projetos escaláveis e aplicações robustas. 


















 







React 



Uma biblioteca JavaScript flexível e de código aberto para o desenvolvimento de interfaces de usuário (UI). Tornou-se uma das principais escolhas para o front-end por permitir que os desenvolvedores trabalhem em um browser visual.
